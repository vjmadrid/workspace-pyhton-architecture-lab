Quickstart¶

The Pokédex package contains basic information about the three starter Pokémon from the core series. These are detailed in API Documentation.

About starter Pokémon¶

As you can see in the Starter Pokémon from the core series. image, these starter Pokémon have different types and abilities, and will evolve into different creatures as their level progresses.

Base Stats¶

The base stats for these Pokémon can be obtained from thi general base stats list. If you need to compute total damage done in battle or the current HP for a given Pokémon, you can use NumPy array objects.

Usage¶

You can create an instance of Bulbasaur called friend, for example, by doing

>>> import pokedex
>>> friend = pokedex.Bulbasaur()
